# Constellation Orbit (p5.js)

A generative constellation field driven by an orbiting attractor instead of the mouse.  
Includes keyboard shortcuts, UI controls, and recording support via MediaRecorder.

## Files
- index.html
- sketch.js
- README.md

## Controls
- Orbit speed / orbit jitter / radius variability sliders  
- **Space** — pause  
- **R** — reset  
- **S** — start recording  
- **E** — end recording and download
